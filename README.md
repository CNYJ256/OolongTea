# 选课社区

同济大学课程评价搜索与浏览，基于「乌龙茶」选课社区数据构建的静态站点。

## 功能

- 🔍 **课程搜索** — 按课程名、教师、代码、院系、分类模糊搜索
- 📊 **可信度排序** — 基于 `评分 × ln(评价数+1)` 排序，避免少量高分霸榜
- 📝 **评价浏览** — 课程详情、成绩分布（优/良/中/及格/不及格）、真实评价
- 🔗 **可分享** — 搜索条件保存在 URL 中，可直接复制分享
- 📱 **移动端适配**

## 技术栈

Vite + Vue 3 + Vue Router + Fuse.js + Tailwind CSS  
GitHub Actions → GitHub Pages 自动部署

## 数据来源

数据来自已停止运营的选课社区「乌龙茶」，由社区成员备份保存。

- **8,451** 门课程 · **7,323** 条评价 · 仅 1 条孤立数据
- 时间跨度：2024/08/02 ~ 2026/05/24

## 本地开发

```bash
npm install
npm run dev          # 开发服务器（需先有 public/data/）
npm run build:data   # 数据预处理（需 wlc.*.json 源文件）
npm run build        # 完整构建（数据预处理 + 前端打包）
```

源数据文件未纳入版本控制。预处理后的 `public/data/` 已提交，`npm run dev` 可直接使用。
